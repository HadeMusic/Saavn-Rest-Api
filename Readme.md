## Saavn Rest Api
This api allows you to bypass the jiosaavn goe-blocking.

### Endpoints

## Get Search
`/saavn/search/query=Faded By Alan Walker`

## Get Track
`/saavn/track/id=...`

## Get Album
`/saavn/album/id=...`

## Get Artist
`/saavn/artist/id=...`

## Get Playlist
`/saavn/playlist/id=...`

## Get AutoComplete
`/saavn/autocomplete/query=...`
